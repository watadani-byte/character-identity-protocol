## How to Reproduce CIP

To reproduce CIP without the provided assets:

1. Generate a high-density reconstruction sample
1. Assign a UID using the provided template
1. Reduce prompt using the Minimal Prompt template
1. Generate a multi-view character sheet
1. Apply Identity Gates (Face ∧ Skeleton ∧ Proportion)

See [Getting Started](../getting_started.md) for full procedure.
