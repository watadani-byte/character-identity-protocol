# UID Template — Character Identity Protocol

Use this template to construct a UID for CIP anchor registration.

-----

## UID Structure

### 1. Identifier

```
Name: [Character name or code]
UID: [Unique identifier string]
```

### 2. Profile

```
Age appearance: [e.g. mid-20s]
Hair: [color, length, style]
Eyes: [color, shape]
Skin tone: [warm / neutral / cool]
Build: [slim / athletic / etc.]
Distinctive features: [e.g. glasses, mole, scar]
```

### 3. Body Proportion

```
Head-to-body ratio: [e.g. 1:7.5]
Shoulder width: [narrow / medium / broad]
Silhouette: [e.g. slim hourglass]
```

### 4. Color Identity

```
Hair color (hex or description):
Eye color (hex or description):
Skin tone (hex or description):
```

### 5. Rendering Constraints

```
Style: [e.g. anime / semi-realistic / realistic]
Line art: [thin / medium / thick]
No facial whitening: [yes / no]
Color temperature: [warm / neutral / cool]
```

### 6. Accessories (Default State)

```
Glasses: [yes / no — type if yes]
Earrings: [yes / no — type if yes]
Necklace: [yes / no — type if yes]
```

-----

> This UID is used for identity reference and gate validation.
> It is not a generation prompt.
